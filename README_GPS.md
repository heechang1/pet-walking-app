# GPS 경로 추적 기능 안내

## 📍 기능 개요

이 앱은 모바일 웹 브라우저에서 GPS를 사용하여 산책 경로를 실시간으로 추적하고 지도에 표시합니다.

## 🚀 주요 기능

### 1. 실시간 경로 추적
- 산책 시작 시 자동으로 GPS 추적 시작
- 5m 이상 이동할 때마다 위치 기록 (노이즈 제거)
- 실시간으로 지도에 경로 표시

### 2. 지도 표시
- **산책 중 페이지**: 실시간 경로와 현재 위치 표시
- **달력 페이지**: 선택한 날짜의 산책 경로 확인
- 시작점(민트색)과 종료점(핑크색) 마커 표시

### 3. 거리 계산
- 총 이동 거리 자동 계산
- 미터/킬로미터 단위로 표시

## 📦 필요한 패키지

```bash
npm install leaflet
npm install --save-dev @types/leaflet
```

## 🔧 사용 방법

### 1. 위치 권한 허용
- 모바일 브라우저에서 위치 권한을 허용해야 합니다
- HTTPS 환경에서만 작동합니다 (로컬 개발은 localhost에서 가능)

### 2. 산책 시작
- `/start` 페이지에서 산책 시작
- `/walking` 페이지로 이동하면 자동으로 GPS 추적 시작
- 지도에 실시간 경로가 표시됩니다

### 3. 경로 확인
- `/calendar` 페이지에서 날짜 클릭
- 산책 기록이 있는 날짜는 경로 지도가 표시됩니다

## ⚠️ 주의사항

1. **HTTPS 필수**: GPS 기능은 HTTPS 환경에서만 작동합니다
   - 로컬 개발: `localhost`는 예외적으로 허용
   - 배포: HTTPS 인증서 필요

2. **모바일 권한**: 모바일 브라우저에서 위치 권한을 허용해야 합니다

3. **배터리 소모**: GPS 추적은 배터리를 많이 소모할 수 있습니다

4. **정확도**: 실내나 건물 근처에서는 GPS 정확도가 떨어질 수 있습니다

## 🗺️ 지도 라이브러리

- **Leaflet**: 오픈소스 지도 라이브러리
- **OpenStreetMap**: 무료 타일 맵 (API 키 불필요)

## 📱 모바일 최적화

- 반응형 디자인으로 모바일 화면에 최적화
- 터치 제스처 지원
- 배터리 효율적인 위치 추적

## 🔒 개인정보 보호

- 모든 위치 데이터는 로컬(localStorage)에만 저장됩니다
- 서버로 전송되지 않습니다
- 사용자가 직접 삭제할 수 있습니다
